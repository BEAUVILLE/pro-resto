# DIGIY MODULE PRO

Ce module fait partie de l’écosystème **DIGIYLYFE**.

## Accès sécurisé

- Entrée via `pin.html`
- Auth universelle : **slug + PIN**
- Vérification Supabase RPC : `digiy_verify_access`
- Session stockée localement (8h)

## Fichiers de base

- `index.html` → redirection vers PIN
- `pin.html` → porte d’entrée sécurisée
- `guard.js` → protège cockpit + pages métier
- `cockpit.html` → tableau de bord principal
